Getting Logisim

Logisim is free software, released under the terms of the GNU General Public License, version 2. It should run on any platform supporting Java, version 5 or later.

  1. Logisim requires Java 5 or later. If you do not already have it on your computer, Java is available from java.sun.com.

  2. Download Logisim from Logisim's SourceForge.net page. You will three choices of which release to download.

    • A .jar file - this should run on any platform, though not necessarily conveniently.
    • A .tar.gz file for MacOS X
    • A .exe file for Windows

    If you use MacOS X or Windows, the release specific to your platform will work best.

  3. To execute the program:

    • With the generic .jar file: On many platforms, you can start the JAR file by double-clicking its icon. If that doesn't work, find your platform's command line and enter:

      java -jar logisim-generic-XX.jar

    • With the MacOS X version: First uncompressed the downloaded .tar.gz file: This might happen automatically, or you might have to double-click it. Then just double-click the Logisim icon to start. You may want to move the icon into the Applications folder.

    • With the Windows version: Logisim does not have an install process; the executable file is truly the file that you want to execute. Just double-click the Logisim icon. You may want to create a shortcut on the desktop and/or in the Start menu to make starting Logisim easier.

  4. The standard release of Logisim includes an English translation (written by TRANSLATOR NAME of TRANSLATOR INSTITUTION). To use the English translation:

    1. After starting Logisim, go to the File menu and select Preferences... (On a Macintosh, this will be in the Logisim menu.)
    2. At the top of the Preferences window, select the International tab.
    3. From the dropdown menu labeled Language, select Russian language.
