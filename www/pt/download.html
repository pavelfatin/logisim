Para obter o Logisim

O Logisim é um software gratuito, liberado sob os termos da GNU General Public License, version 2.

É esperado que seja executado em qualquer plataforma que ofereça suporte à linguagem Java, versão 5 ou superior.

  1. O Logisim requer Java 5 ou superior. Se você não a tiver em seu computador, Java é disponível a partir de Oracle/Java.

  2. Download Logisim a partir da párgina Logisim's SourceForge.net. Você terá três escolhas sobre as versões a serem baixadas.

    • Um arquivo .jar - esse deve ser executado em qualquer plataforma, embora não seja de todo conveniente.
    • Um arquivo .tar.gz para MacOS X
    • Um arquivo .exe para Windows

    Se você usar MacOS X or Windows, a versão específica para a sua plataforma deverá funcionar melhor.

  3. Para executar o programa:

    • Com o arquivo genérico .jar: Na maioria das plataformas, você poderá iniciar o arquivo JAR ao usar um duplo-clique sobre o seu ícone. Se isso não funcionar, usar a linha de comando da sua plataforma e digitar:

      • java -jar logisim-generic-XX.jar

    • Com a versão para MacOS X: primeiro, expandir o arquivo .tar.gz: Isso deverá ocorrer automaticamente, ou você precisará dar um duplo-clique sobre ele. Basta um duplo-clique sobre o ícone do Logisim para esse ser iniciado. Você poderá mover o ícone para a pasta de Aplicações.

    • Com a versão para Windows: o Logisim não possui um processo de instalação; o arquivo executável é, na verdade, o arquivo que você deseja executar. Basta um duplo-clique sobre o ícone do Logisim. Você poderá criar um atalho para a área de trabalho e/ou para a o menu Iniciar para lançá-lo de modo mais fácil.

Para usar a tradução para português do Brasil

A versão padrão do Logisim inclui uma tradução para língua portuguesa (escrita por NOME DO TRADUTOR da INSTITUIÇÃO DO TRADUTOR). Para usar a tradução para português do Brasil:

  1. Após iniciar o Logisim, ir até o menu File e selecionar Preferences... (Em um Macintosh, isso estará no menu do Logisim.)
  2. Ao topo da janela de Preferences, selecionar a guia International.
  3. A partir do menu dropdown onde se lê Language, selecionar Portuguese.
